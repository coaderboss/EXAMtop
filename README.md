## 🧠 Core Architecture & Innovations

ExamiTop is not just a UI; it is a highly optimized execution environment built to solve the most complex problems in digital academic testing.

### 1. Flicker-Free MathJax V3 Engine
Most platforms struggle with raw LaTeX flashing on screen. We completely re-engineered the rendering pipeline. By injecting smart React references and utilizing MathJax Promise Queues, complex calculus and physics equations render synchronously and flawlessly without exposing the underlying code.

### 2. Zero-Tolerance Proctoring Sentinel
A background daemon actively monitors the testing environment. It tracks `visibilitychange` events, full-screen evasions, and clipboard shortcuts. Every single violation is cryptographically timestamped and instantly relayed to the Examiner Vault for complete session transparency.

### 3. Double-Negative Safe Scoring Logic
Our evaluation engine uses absolute-value sanitization to prevent the notorious double-negative integer bug. It flawlessly evaluates Multiple Select Questions (MSQ) with partial marking logic, un-attempted arrays, and MCQ negative penalties with perfect mathematical precision (O(n) time complexity).

### 4. True Offline Vault (Device-Native Storage)
Network failure during exams is no longer a catastrophe. ExamiTop utilizes browser-level `IndexedDB` and `localStorage` APIs to create, conduct, and evaluate entire test series completely locally. Once the network is restored, the encrypted payload seamlessly syncs with the central database.

## 🔐 System Modules

The platform is strictly divided into highly secure clearance levels:

### 🎓 Student Terminal
* **Distraction-Free Interface:** Full-screen enforced UI for maximum focus.
* **Smart Question Palette:** Real-time color-coded tracking (Attempted, Marked for Review, Skipped).
* **Virtual Rough Pad:** Integrated floating scratchpad for mathematical calculations.
* **Live Analytics:** Instant post-exam scorecard with percentage accuracy and detailed question-wise review.

### 💼 Examiner Vault
* **Intelligent Test Creator:** Formulate MCQ, MSQ, Integer, and Subjective questions with rich text and mathematical equations.
* **Smart Shuffle Algorithm:** Fisher-Yates powered shuffling that intelligently anchors options like *"All of the above"*.
* **Deep Telemetry:** Monitor live test intakes, review timestamped cheat logs, and manage negative marking matrices.

### 👑 Omni-Control Center (God Mode)
* **Live System Radar:** Real-time monitoring of all platform activity and submissions.
* **Citizen Matrix:** Manage clearance levels, suspend suspicious accounts, and perform database wipes.
* **God Voice:** Broadcast emergency system updates instantly to all connected clients.
* **Graphical Analytics:** Doughnut and Line charts visualizing global platform accuracy and 7-day engagement trends.
